# IMC.PYTHON

O projeto é um código que pode ser  calculado o IMC, quando o usuário informa sua altura e peso, depois o código calcula e exibe o IMC ao usuário. Outra detalhe, além de calcular o IMC, o código exibe a classificação do IMC do usuário, como "Abaixo do Peso", "Obesidade Grau I".
